Politica de privacidad

Esta politica explica como ByFlow recopila, usa, comparte y protege datos personales cuando utilizas el sitio web, la PWA y las herramientas web de ByFlow, incluyendo funciones de karaoke, cuentas, pagos, analitica y publicidad.

1. Responsable

Responsable del tratamiento: Arturo Torres / IArtLabs.

Contacto de privacidad y soporte: elricondelgeekdearturo@gmail.com.

2. Datos que podemos recopilar

  • Datos de cuenta y autenticacion, como nombre, correo electronico, UID y foto de perfil cuando usas Firebase Auth o inicios de sesion sociales.
  • Datos operativos del servicio, como letras cargadas, cola de karaoke, preferencias, estadisticas de uso, configuraciones y eventos dentro de la app.
  • Datos de pago y facturacion procesados por Stripe. ByFlow no almacena numeros completos de tarjeta en sus paginas.
  • Datos tecnicos y de dispositivo, como navegador, sistema operativo, idioma, IP aproximada, identificadores de sesion, cookies y eventos de rendimiento.
  • Datos de publicidad y medicion cuando se habiliten Google Analytics y Google AdSense o herramientas similares.

3. Finalidades de uso

  • Prestar, mantener y mejorar ByFlow y sus funciones de karaoke, teleprompter, musica, cuentas, suscripciones y analitica.
  • Gestionar acceso, seguridad, sesiones, soporte tecnico, recuperacion de errores y prevencion de fraude o abuso.
  • Procesar pagos, activaciones, licencias, suscripciones, comprobaciones y comunicaciones relacionadas con la cuenta.
  • Medir uso del producto, conversiones y rendimiento comercial cuando se habilite Google Analytics.
  • Mostrar publicidad contextual o personalizada cuando se habilite Google AdSense o servicios equivalentes.
  • Cumplir obligaciones legales, regulatorias, fiscales o contractuales.

4. Bases legales

Dependiendo del caso, tratamos datos para ejecutar el servicio solicitado por el usuario, cumplir obligaciones legales, proteger intereses legitimos de seguridad y operacion, o con base en consentimiento cuando aplique para cookies, analitica o publicidad.

5. Terceros y proveedores

ByFlow puede compartir datos solo en la medida necesaria con proveedores que ayudan a operar la plataforma, por ejemplo:

  • Google Firebase para autenticacion y servicios relacionados.
  • Stripe para cobros, suscripciones, validaciones y facturacion.
  • Google Analytics para medicion de trafico y eventos.
  • Google AdSense para monetizacion publicitaria, si se activa.
  • Infraestructura, hosting, monitoreo, correo o soporte tecnico.
  • Autoridades o terceros cuando la ley lo requiera o sea necesario para proteger derechos y seguridad.

6. Cookies, analitica y publicidad

ByFlow puede usar cookies, almacenamiento local y tecnologias similares para recordar sesiones, preferencias, rendimiento, seguridad y medicion.

Si Google Analytics esta habilitado, se podran registrar visitas, paginas vistas, eventos y datos agregados de uso. Si Google AdSense esta habilitado, terceros podran usar cookies para mostrar anuncios y medir resultados.

El usuario puede bloquear cookies desde su navegador, aunque ciertas funciones podrian dejar de operar correctamente.

7. Conservacion de datos

Conservamos datos solo durante el tiempo necesario para operar el servicio, atender soporte, cumplir obligaciones legales, resolver disputas y hacer valer acuerdos. Cuando ya no son necesarios, pueden ser eliminados, anonimizados o agregados.

8. Seguridad

Aplicamos medidas tecnicas y organizativas razonables para reducir riesgos de acceso no autorizado, perdida, alteracion o divulgacion. Ningun sistema conectado a internet es completamente infalible, por lo que no podemos garantizar seguridad absoluta.

9. Derechos del usuario

Podras solicitar acceso, rectificacion, cancelacion, oposicion, eliminacion o aclaracion sobre el tratamiento de tus datos, sujeto a requisitos legales y tecnicos aplicables. Para ello, escribe a elricondelgeekdearturo@gmail.com.

10. Menores de edad

ByFlow no esta dirigido de forma principal a menores de edad para contratacion comercial. Si detectamos datos de menores tratados sin base valida, podremos eliminarlos o restringir el servicio.

11. Cambios a esta politica

Podemos actualizar esta politica para reflejar cambios legales, operativos o comerciales. La version vigente sera la publicada en esta URL con su fecha de actualizacion.

Si necesitas una version contractual o corporativa para cumplimiento interno, onboarding comercial o revision de terceros, puedes solicitarla por correo.
